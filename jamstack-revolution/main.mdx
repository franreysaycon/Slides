# JAMSTACK 

![JAMStack Diagram](https://i.ibb.co/dj9GD7X/JAMSTACK.png)

---

## JAMSTACK MAIN TAKEAWAYS
1. Javascript is already built.
2. Whole frontend is hosted on a CDN. 
3. All business operations is done through an API.
4. The view is purely a view.  

---

## JAMSTACK ADVANTAGES

1. Static - no need for SSR, already built
2. Improved Scaling 
3. Low Operating Costs
4. Accomodating to slower internet
5. Improved Developer Experience
6. Better Security

---

## JAMSTACK DISADVANTAGES

1. Serverless contact
2. Pre-built
